# customwordcount

* wc is a linux command line tool. This project implements certain functionalities of wc in java.

Implementation
* Using maven-assembly plugin to generate a uber jar.
* How to run?
  * update bash/ccwc file.
  * move this file to /usr/local/bin
  * How to run: ccwc -c file.txt
